🔒 隐私与安全

📊

数据收集

我们仅收集以下数据用于改进产品：

  • 使用统计：命令使用频率、功能使用情况
  • 错误日志：用于问题诊断和修复

我们不会收集您的源代码内容或敏感信息。

🛡️

隐私保护

  • 本地优先：代码分析和处理优先在本地进行
  • 数据加密：所有网络传输数据均经过加密
  • 最小化原则：仅在必要时传输最少的数据
  • 透明度：明确告知数据使用方式

🤝 社区与支持

🐛

反馈问题

  • GitHub Issues：在 Issues 页面 中提交详细问题报告
  • 技术支持：通过官方邮箱联系 codebuddy@tencent.com

提交问题时，请包含以下信息：

  • 操作系统和版本
  • Node.js 版本
  • CodeBuddy Code 版本
  • 详细的错误描述和复现步骤
📚

获取帮助

  • 内置帮助：使用 /help 命令获取即时帮助
  • 命令参考：使用 codebuddy --help 查看所有可用选项
  • 系统诊断：使用 /doctor 检查系统状态

📝 更新日志

查看 CHANGELOG.md 了解详细的版本更新记录，包括：

  • 新功能和改进
  • Bug 修复
  • 破坏性变更
  • 性能优化

我们建议定期更新到最新版本以获得最佳体验和安全保障。
